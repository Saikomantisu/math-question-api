# Math Question API

### Endpoints
`/api` - Get all the questions

| Key | Value | Description |
| ----------- | ----------- | ----------- |
| `level` | ["easy", "medium", "hard"] | Get question based on the level
| `limit` | 1 - 3550 | Limit the question array

`/api/random` - Get a random question

| Key | Value | Description |
| ----------- | ----------- | ----------- |
| `level` | ["easy", "medium", "hard"] | Get question based on the level
